# DevOps & SecOps Labs

A collection of hands-on laboratories for learning and experimenting with technologies across the DevOps to SecOps spectrum.

## Purpose

This repository serves as a personal learning space to explore, test, and understand modern infrastructure, automation, security, and operations tools and practices.

## Labs

| Lab | Description | Key Technologies |
|-----|-------------|------------------|
| [Zero-Touch Kubernetes on DigitalOcean](./zero-touch-kubernestes-digital-ocean/) | Full GitOps platform with observability stack and multi-tier applications | Kubernetes, ArgoCD, Terraform/OpenTofu, Prometheus, Loki, Grafana, Go, Python, Rust |

## Structure

Each lab is self-contained in its own directory with:
- A detailed `README.md` with setup instructions and architecture overview
- All necessary source code and configuration files
- Infrastructure as Code definitions where applicable

## Getting Started

Navigate to any lab directory and follow the instructions in its README to get started.
